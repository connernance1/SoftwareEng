# SnapShare

## Purpose

- To List Media a user owns that they can add to, delete from, and 
  share to other users on the same platform.

## Name Schema & Organization

- Ability to see other Users Movies/Data
- Add, Update, Delete Listss of Movies/Data
- Movie list per user that is searchable 



## Install from Git

Create own [Git](https://git-scm.com/downloads) Repository and add remote repository 

```bash
git init

git remote add origin https://github.com/usu-cs-3450/Repo-2.2.git
```
